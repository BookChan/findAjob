# 收集信息的方式
1.问有经验的人
2.找行业研究报告
3.搜索零碎信息,自己整理回答


# Ask right question
X = your expected job 
- Q1: Is X  an important discipline?
- Q2: Where is X most needed?
- Q3: How is X as a career?
- Q4: What do I need to learn for preparing myself?
- Q5: How shall I distinguish myself in X?
